# Pixel Mock Studio 📱✨

**Live Demo:** [https://daklok.github.io/Phone-Mockup/](https://daklok.github.io/Phone-Mockup/)

A sleek, 100% client-side web application to generate beautiful Google Pixel 10 device mockups from your photos and videos. Built with pure HTML, CSS, and Vanilla JavaScript, following the **Material 3 Expressive** design guidelines.

> **⚠️ IMPORTANT NOTICE:**
> While the application can preview video files, **video export functionality is currently experimental and not yet fully supported.** You may experience unexpected behavior, browser-specific limitations, or performance drops when trying to render and download video mockups. Static image (PNG) exports work perfectly.

## ✨ Features

* **100% Client-Side Processing**: Everything happens directly in your browser. No media is ever uploaded to a server, ensuring absolute privacy and fast processing.
* **Material 3 Expressive UI**: Features a beautiful dark-mode interface with spring-based animations, ambient floating gradients, and custom M3 color tokens.
* **Multiple Device Colors**: Choose from authentic Pixel 10 frame finishes:
  * Obsidian 🖤
  * Porcelain 🤍
  * Rose 🌸
  * Hazel 🌿
* **Dynamic Perspectives**: Generate flat front-facing mockups or dynamic 3D angled perspectives (Left & Right).
* **Drag & Drop Interface**: Easily upload multiple files at once.
* **Zero Dependencies**: No external libraries, no NPM packages, and no build steps required.
